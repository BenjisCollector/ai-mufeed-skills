// 01

شو هي Skills ببساطة

تخيّل إجاك موظف جديد شاطر. أول يوم، بتقعد معه ربع ساعة وبتشرحلو: "هيك بدي التقارير، بهاللهجة بردّ على الإيميلات، وهاي الأشياء انتبه إلها."

الموظف الشاطر بحفظ كلامك. بعدها، مش لازم تعيد الشرح كل يوم. بشتغل الشغل بطريقتك لحالو، وإنت بس بتراجع.

Skills هي بالظبط هالإشي

Skill يعني «مهارة». إنت بتكتب لـ Claude تعليماتك مرة وحدة (كيف بدك الشغل)، وهو بحفظها عندو. بعد هيك، كل ما تطلب منه إشي من هالنوع، بيرجعلها لحالو ويشتغل بطريقتك، بدون ما تعيد الشرح ولا مرة.

يعني بدل ما تكون كل يوم بتشرح من الأول لواحد نساك، صار عندك واحد بتذكّر كل تفاصيلك ويشتغل عليها بروحو.

// 02

ليش بتوفّرلك ساعات؟

لأنو الوقت اللي بيضيع مش بالشغل نفسه. الوقت بيضيع إنك تعيد شرح نفس الإشي كل مرة.

بدون Skills
  • كل صبح بتفتح شات جديد وبتشرح من الصفر
  • بتكتب نفس التعليمات: "خلّيها رسمية، قصيرة، بالعربي..."
  • بتنسى تفصيل، فالنتيجة بتطلع غلط وبترجّعها
  • كل مرة النتيجة بتطلع بشكل مختلف
مع Skills
  • بتكتب تعليماتك مرة وحدة وبتنساها
  • Claude بيمسكها لحالو كل مرة بدون تذكير
  • النتيجة بتطلع بطريقتك من أول طلب
  • نفس الجودة كل مرة، ثابتة

الفرق بسيط بس ضخم: إذا في شغلة بتعملها أكتر من مرة بالأسبوع (رد إيميلات، تلخيص اجتماعات، مراجعة أرقام)، هاي بالظبط اللي Skill بتاخدها عن كتافك.

// 03

كيف تعمل Skill، خطوة بخطوة

قبل ما تبدأ، خليني أطمنك: مش لازم تعرف برمجة ولا إشي تقني. كلو بالكلام العادي داخل تطبيق Claude. خطوة بخطوة:

1

شغّل خاصية «تشغيل الأكواد»

Skills ما بتشتغل إلا إذا فعّلت خاصية اسمها Code execution. روح على Settings › Capabilities وشغّل الزر اللي مكتوب عليه «Code execution and file creation». مرة وحدة بس وبتنساها.

2

افتح قسم Skills

من نفس الإعدادات، روح على Customize › Skills. هون بتلاقي مكان كل الـ Skills، وبتقدر تشغّل أو تطفّي أي وحدة، وتضيف وحدة جديدة.

3

اضغط «Create» وخلّي Claude يبنيها معك

لما تضغط زر الإضافة، رح يطلعلك خيار «Create with Claude». هاي أسهل طريقة: بتحكي مع Claude بالكلام العادي، بتقلّو شو بدك تعمل Skill لإيش، وهو بسألك أسئلة ويكتبلك التعليمات لحالو. إنت بس بتجاوب.

4

احكيلو التعليمات بلغتك

كل Skill لازم يكون إلها اسم (قصير) ووصف بسطر واحد بيقول إمتى تُستخدم: مثلاً: «راجع فواتير المصاريف وطلّع اللي غلط». هاد الوصف مهم لأنو منه Claude بيعرف لحالو إمتى يستعمل الـ Skill. بعدها بتكتب جوّا التفاصيل: شو بدك بالظبط، وشو ينتبهلو، وشكل النتيجة اللي بترضيك.

5

جرّبها، وخلّيها تشتغل لحالها

بعد ما تحفظها، ارجع على الشات العادي واطلب طلبك بشكل طبيعي. مش لازم تنادي على الـ Skill ولا تكتب اسمها. Claude بيعرف لحالو إنو هاد الطلب إلو Skill، وبيستعملها بروحو. إذا النتيجة بدها تعديل، احكيلو، وبيظبط الـ Skill معك بنفس الشات.

في طريقة تانية للمتقدمين (اختيارية)

لو حابب، بتقدر تكتب التعليمات بملف اسمه SKILL.md جوّا مجلد، وتضغطه ملف ZIP وترفعو من نفس صفحة Customize › Skills. بس بصراحة، لأغلب الناس، خيار «Create with Claude» بالكلام أسهل وأسرع. ما في داعي تعقّد حالك.

قبل ما نكمّل، إذا هالإشي فتحلك عينك، اعمل Follow لـ @yusufmufeed.exe عشان يوصلك كل شغلة بتوفّرلك وقت بشغلك.
// 04

شو بيروح جوّا الـ Skill؟

عشان تطلع Skill منيحة، فكّر فيها كأنك بتكتب ورقة تعليمات لموظف جديد. أربع أشياء بس:

  • اسم واضح: شو اسم المهارة (مثلاً: «رد الإيميلات بلهجتي»).
  • وصف بيقول إمتى تُستعمل: هاد اللي منه Claude بيقرر لحالو يستعملها. خلّيه دقيق.
  • التعليمات: شو بدك بالظبط، خطوة خطوة، وشو ينتبهلو.
  • مثال على النتيجة (اختياري بس بيفرق كتير)، ورجيه شكل الشغل الصح مرة وحدة، فبيقلّدو.

القاعدة الذهبية: كل ما كان الوصف والتعليمات أوضح، كل ما Claude عرف يستعمل الـ Skill بالوقت الصح ويطلّع نتيجة أقرب لطريقتك. اكتبها زي ما بتشرح لإنسان، مش لجهاز.

// 05

٥ Skills جاهزة انسخها زي ما هي

هاي تعليمات جاهزة كتبتها إلك. افتح «Create with Claude»، الصق النص، وعدّل الأسماء والتفاصيل عشغلك. بلّش فيهم اليوم.

مراجعة فواتير المصاريف

للمحاسب

بتراجع كشوفات المصاريف بإيدك وبتخاف يفوتك خطأ؟ خلّي هاي الـ Skill تمسكهم قبلك.

انسخ هذاالاسم: مراجعة المصاريف الوصف: استعملها كل ما أعطيك جدول أو كشف مصاريف عشان تراجعه وتطلّعلي المشاكل. التعليمات: راجع كل بند بالكشف ودوّر على: - أي مبلغ مكرر أو انصرف مرتين لنفس الشي. - أي رقم أعلى بشكل غريب من عادته (قفزة مش طبيعية). - أي بند ناقصه فاتورة أو وصف واضح. - أي مصروف مش مسموح فيه حسب سياستنا. رتّبلي النتيجة بجدول قصير: البند، المبلغ، وشو المشكلة، وكلمة وحدة: (سليم / لازم أتأكد / غلط). لا تخترع أرقام، إذا مش متأكد قول «لازم أتأكد».

رد الإيميلات بلهجتي

للكل

تعبت تعيد كتابة نفس نوع الردود؟ خلّي Claude يرد بطريقتك إنت، مش بطريقة روبوت.

انسخ هذاالاسم: ردودي على الإيميل الوصف: استعملها كل ما ألصقلك إيميل وأطلب رد. التعليمات: اكتب الرد بأسلوبي: - مهذّب بس مختصر، بدون حشو ولا مجاملات طويلة. - ابدأ بالمهم على طول، بدون مقدمات فاضية. - استعمل جُمل قصيرة وواضحة، عربي فصيح بسيط. - إذا في طلب، لخّصه بنقطة وحدة بالآخر عشان يبيّن. - خليك دافي بس محترف، لا رسمي ناشف ولا كتير ودّي. اكتبلي نسخة وحدة جاهزة للإرسال، وإذا في جملة حساسة، نبّهني عليها.

من محضر الاجتماع لمهام

لمدير الفريق

بتطلع من الاجتماع وراسك مليان، وبتنسى مين لازم يعمل شو؟ هاي بتحوّل الحكي لمهام واضحة.

انسخ هذاالاسم: مهام الاجتماع الوصف: استعملها كل ما ألصقلك ملاحظات أو محضر اجتماع. التعليمات: اقرا الملاحظات وطلّعلي: - قائمة مهام واضحة، كل مهمة بجملة تبدأ بفعل (جهّز، ارسل، راجع...). - جنب كل مهمة: مين المسؤول (إذا مذكور) وإمتى الموعد (إذا مذكور). - قسم منفصل صغير: القرارات اللي اتفقنا عليها. - قسم صغير: نقاط لازمها متابعة أو لسا مش محسومة. خليها مختصرة ونقاط، مش فقرات طويلة. لا تضيف مهام من عندك، بس اللي انذكر.

كتابة المحتوى بصوت البراند

للتسويق

كل ما حدا يكتب بوست، بيطلع بصوت مختلف؟ ثبّت صوت البراند بـ Skill وحدة.

انسخ هذاالاسم: صوت البراند الوصف: استعملها كل ما أطلب منك كتابة بوست أو كابشن أو إعلان. التعليمات: اكتب بصوت براندنا: - نبرة: (اكتب هون نبرتك، مثلاً: بسيطة، جريئة، بتحكي لشخص واحد). - ممنوعات: لا كلام رسمي ناشف، لا مصطلحات معقّدة، لا مبالغة. - كل فكرة بجملة قصيرة، وحدة بتجرّ اللي بعدها. - ابدأ دايماً بالفايدة اللي بتوصل للقارئ، مش بالميزة التقنية. - اختم بخطوة وحدة واضحة يعملها القارئ. اعطيني ٣ نسخ مختلفة أختار منها، وقلّي أي وحدة أقوى وليش.

التقرير الأسبوعي

للموظف / المدير

آخر الأسبوع وبتقعد ساعة تجمّع شو عملت؟ خلّيها تنكتب لحالها بنفس الشكل كل مرة.

انسخ هذاالاسم: تقريري الأسبوعي الوصف: استعملها كل ما ألصقلك نقاطي أو شغل الأسبوع وأطلب تقرير. التعليمات: حوّل نقاطي لتقرير أسبوعي مرتّب بنفس الشكل كل مرة: - عنوان: أهم إنجاز هالأسبوع بسطر واحد. - «خلصنا»: نقاط للشغل اللي انعمل. - «شغّالين عليه»: نقاط للشغل الجاري. - «عالق / بدو قرار»: أي إشي واقف ومحتاج مساعدة. - بالآخر: خطة الأسبوع الجاي بـ ٣ نقاط. خليه مختصر، مهني، ونقاط واضحة. لا تكبّر الإنجازات ولا تخترع إشي.

أخطاء شائعة ونصائح

أخطاء لا تعملها
  • وصف غامض («ساعدني بالشغل»)، Claude ما بيعرف إمتى يستعملها.
  • تحشر ١٠ مهام مختلفة بـ Skill وحدة، كل مهارة لحالها.
  • تنسى تفعّل Code execution، فبتضل مطفية وما تشتغل.
  • تكتب تعليمات مبهمة وتتوقع نتيجة دقيقة.
نصائح تظبطها
  • Skill وحدة = مهمة وحدة واضحة. بسيطة بتشتغل أحسن.
  • احكيلو «لا تخترع أرقام» عشان يضل صادق ودقيق.
  • ورجيه مثال على النتيجة الصح مرة وحدة، بيقلّدو.
  • عدّل الـ Skill بعد أول تجربة، أول نسخة مش لازم تكون كاملة.

ملاحظة صادقة عن الخطط: خاصية Skills متوفرة على كل الخطط، المجاني (Free)، و Pro، و Max، و Team، و Enterprise. بس لازم تفعّل خاصية «Code execution and file creation» الأول، لأنها بدونها ما بتشتغل. هاي معلومة صحيحة وقت كتابة هالدليل، وممكن Claude يطوّرها بعدين.
